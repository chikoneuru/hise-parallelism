# Carbon intensity traces

Two ways to feed HISE's control loop:

1. **Synthetic** — `synthetic_solar.csv` is generated by
   `hise.energy.carbon_trace.synthetic_solar_trace()` and committed for deterministic CI.
   24h cycle, 5-minute samples, sinusoidal solar pattern (low at noon, high overnight).
2. **Real** — drop ElectricityMaps "history" CSV exports here (any zone). Schema:
   `timestamp,intensity_g_per_kwh` (ISO-8601 timestamps, gCO2/kWh values).
   Point `HISE_CARBON_TRACE` at the file you want to replay.

Recommended zones for the planned evaluation (mix of clean + dirty grid):

| Zone | Reason |
|---|---|
| `IS` (Iceland) | extreme low intensity (~20 gCO2/kWh, geothermal-dominated) |
| `PL` (Poland) | extreme high (~600–800 gCO2/kWh, coal-heavy) |
| `DE` (Germany) | high diurnal swing (solar+wind mix) |
| `VN` (Vietnam) | candidate's home region for CEI deployment story |

For the API path, set `ELECTRICITYMAPS_API_KEY` and `ELECTRICITYMAPS_ZONE` and pass
`--use-api` to the experiment scripts.
